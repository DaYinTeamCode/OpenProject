<h2 id="1.0.0"> 一.组件化方案</h2>

### 1.得到组件化方案
---

- 组件可以单独调试
- 杜绝组件之前相互耦合，代码完全隔离，彻底解耦
- 组件之间通过接口+实现的方式进行数据传输
- 使用scheme和host路由的方式进行activity之间的跳转
- 自动生成路由跳转路由表
- 任意组件可以充当host，集成其他组件进行集成调试
- 可以动态对已集成的组件进行加载和卸载
- 支持kotlin组件

[得到组件化方案](https://github.com/luojilab/DDComponentForAndroid)
